== Description ==
O **SEO PT** ajuda a otimizar conteúdos para Português (PT-PT e PT-BR), com revisão linguística, boas práticas de SEO e uma base de dados de localização para Portugal.

**Funcionalidades principais**
* Conversão PT-BR ⇄ PT-PT com dicionários personalizados
* Revisão de sintaxe e estilo (alerta de gerúndios, redundâncias, etc.)
* Módulo de localização com distritos/concelhos/localidades (JSON)
* Interface simples no WP Admin (abas: Geral / Localização)

== Installation ==
1. Carrega a pasta do plugin para `wp-content/plugins/seo-pt/`
2. Ativa o plugin em **Plugins → Installed Plugins**
3. Acede ao menu **SEO PT** no Admin para configurar

== Frequently Asked Questions ==
= Este plugin funciona com Gutenberg? =
Planeada a integração (roadmap).

= Posso adicionar palavras personalizadas ao dicionário? =
Sim, através dos ficheiros JSON em `modules/lingua/dictionary/`.
